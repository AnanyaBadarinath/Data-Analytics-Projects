Objective:
Analyze global restaurant data and build an interactive Power BI report to help evaluate restaurant performance across countries, cities, and cuisines.

Tools Used:
Power BI, Excel, DAX

Key Tasks Performed:

1. Cleaned and transformed restaurant data from multiple Excel files using Power Query

2. Created calculated columns and measures such as restaurant count, average cost, and average rating using DAX

3. Built interactive dashboards with filters for continent, country, city, and service types (online ordering/reservation)

4. Designed a multi-page report with drill-down capability to analyze restaurant ratings, costs, and cuisines

Outcome:
The dashboard enables users to explore restaurant distribution worldwide, identify highly rated restaurants, and analyze cost and cuisine trends across locations.
