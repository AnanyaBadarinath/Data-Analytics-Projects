Restaurant Data Analysis – Data Analyst Capstone Project

Objective:
Analyze restaurant dataset to identify key factors influencing restaurant performance and generate insights that help identify top-performing ("star") restaurants.

Tools Used:
Python, Pandas, Data Visualization, Tableau, Excel

Dataset:
The project uses restaurant dataset containing information about restaurant locations, cuisines, ratings, delivery options, and cost information.

Key Tasks Performed:

1. Performed data cleaning and preprocessing to handle missing values and duplicates

2. Conducted Exploratory Data Analysis (EDA) to understand restaurant distribution across cities and countries

3. Analyzed key factors such as online delivery, table booking, cuisines, and cost affecting restaurant ratings

4. Identified top cuisines, restaurant distribution, and customer voting patterns

5. Built Tableau dashboard to visualize insights and identify potential star restaurants

Outcome:
The analysis provides insights into restaurant performance, customer preferences, and factors influencing ratings, helping businesses identify high-performing restaurants and improve decision making.
