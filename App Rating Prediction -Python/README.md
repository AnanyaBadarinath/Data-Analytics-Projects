App Rating Prediction – Python

Objective:
Build a machine learning model to predict Google Play Store app ratings using various app attributes such as reviews, installs, size, price, and category.

Tools Used:
Python, Pandas, NumPy, Matplotlib/Seaborn, Scikit-learn

Key Tasks Performed:

1. Performed data cleaning and preprocessing by handling null values, correcting data types, and removing inconsistent records

2. Conducted Exploratory Data Analysis (EDA) using histograms, boxplots, and scatter plots to understand relationships between rating and other variables

3. Applied log transformation and dummy encoding to prepare the dataset for modeling

4. Built a Linear Regression model to predict app ratings and evaluated performance using R² score

Outcome:
The model helps identify key factors affecting app ratings and can assist in predicting which apps may perform well on the Google Play Store.
