Sales Dashboard Analysis – Excel

Objective:
Analyze company sales data and create an interactive Excel dashboard to track sales performance across different product categories.

Tools Used:
Microsoft Excel (Pivot Tables, Charts, Data Formatting)

Key Tasks Performed:
1. Analyzed sales data to identify trends across product categories and monthly performance

2. Created pivot tables and charts to summarize and visualize sales data

3. Built an interactive sales dashboard allowing users to filter and view sales by product category and month

Outcome:
The dashboard provides a clear visualization of sales trends, helping users quickly analyze product performance and identify patterns across categories.
