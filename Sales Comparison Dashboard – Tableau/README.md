Sales Comparison Dashboard – Tableau

Objective:
Create a dashboard to compare sales performance between two regions.

Tools Used:
Tableau, Sample Superstore Dataset

Key Tasks:
1. Created parameters to compare Primary Region and Secondary Region
2. Built calculated fields to analyze sales metrics
3. Developed an interactive dashboard showing sales, orders, and customers
4. Organized location data using hierarchies

Outcome:
The dashboard helps management compare regional sales performance and identify areas for improvement.
